# แผนการจัดวางโครงสร้างและดีไซน์หน้าผู้ดูแลระบบรูปแบบใหม่ (Light Theme, Resizable Sidebar, Playlist Grid View, Full Div Height)

แผนงานนี้จัดตั้งขึ้นเพื่อทำการปรับโฉมอินเตอร์เฟสผู้ดูแลระบบ (/admin) ใหม่ทั้งหมด ให้เป็นธีมสว่างสไตล์มินิมัลลิสต์ระดับพรีเมียม (Premium Light Theme Dashboard) ที่คล้ายคลึงกับรูปแบบ Layout ของเครื่องเล่นเพลงสากลชั้นนำ (คล้ายรูปตัวอย่างการวางแถบควบคุมและแผงไลบรารี) พร้อมทั้งรองรับคุณสมบัติการย่อ-ขยายและสไลด์ปิดแถบข้าง (Resizable & Toggleable Sidebar) และจัดหน้าปกแบบ Grid Thumbnail ทั้งในส่วนคลังเพลย์ลิสต์และแทร็กเพลง รวมทั้งปรับความโค้งมนของภาพประกอบทั้งหมดในระบบให้เป็นแบบมนปกติ (rounded)

---

## หัวข้อที่ต้องการการรีวิวจากผู้ใช้งาน (User Review Required)

> [!NOTE]
> - **การใช้ธีมสว่างพรีเมียม (Light Theme):** หน้าจอ `/admin` จะเปลี่ยนสีพื้นหลังและสีส่วนควบคุมหลักทั้งหมดให้เป็นธีมสว่างที่ดูสะอาดตา สไตล์ SaaS สมัยใหม่ และแยกสัดส่วนการเรืองแสงอ่อนๆ ของมรกตไว้อย่างเรียบหรู
> - **คุณสมบัติการย่อ-ขยายแถบข้าง (Resizable & Toggleable Sidebar):** สามารถลากเส้นแบ่งแนวตั้งตรงกลางเพื่อปรับความกว้างของเมนูซ้ายได้ตั้งแต่ 240px ถึง 480px และกดปุ่มป๊อปอัปเพื่อเปิด/ปิด แผงควบคุมทั้งหมดได้อย่างเป็นอิสระ
> - **การปรับแต่งดีไซน์ภาพประกอบ (Rounded Images):** รูปภาพทั้งหมดในระบบ ทั้งหน้าเล่นเพลง ฝั่งลูกค้า และหลังบ้าน จะถูกปรับลดระดับความโค้งมนลงให้เหลือเพียงความมนสากลธรรมดา (`rounded`) เพื่อความคมชัดเป็นระเบียบ

### 3. แผงควบคุมผู้ดูแลระบบหลัก (Admin Dashboard Console Upgrade)

- **การแสดงผลแทร็กเพลงและค้นหาแบบ Full Height & View Selector:**
  - ขยายความสูงของช่องแสดงผลเพลงและผลการค้นหาให้ขยายตัวเต็มพื้นที่ว่างของหน้าจอเบราว์เซอร์ (`flex-1 min-h-0 h-[calc(100vh-250px)]` ร่วมกับ ScrollArea)
  - เพิ่มปุ่มสลับมุมมองระหว่าง **"มุมมองรายการ (List View)"** และ **"มุมมองตารางปกย่อ (Grid Thumbnail View)"** สำหรับแทร็กเพลงภายในเพลย์ลิสต์และผลลัพธ์การค้นหาจาก YouTube
  - รูปภาพปกทั้งหมดในระบบผู้ดูแลระบบจะถูกบีบขอบคมชัดแบบ `rounded` ธรรมดาเท่านั้น

## แผนการตรวจสอบความถูกต้อง (Verification Plan)

### การคอมไพล์โปรแกรม (Build Verification)
- รันคำสั่ง `bun run build` หรือ `npm run build` เพื่อตรวจสอบประสิทธิภาพการแปลและยืนยันว่าไม่มีข้อขัดข้องในโครงสร้าง TypeScript

### การตรวจสอบการควบคุมและลักษณะที่ปรากฏ (UI & Layout Verification)
1. เข้าชมหน้าผู้ดูแลระบบ ตรวจสอบธีมหลักว่าเป็น Light Theme สีสว่าง สะอาดตา คมชัดทุกรายละเอียด
2. ทดลองขยับยืดขยาย Sidebar โดยนำเมาส์ไปวางที่แถบแบ่งแนวตั้งและลากเมาส์ไปทางซ้าย/ขวา ตรวจสอบว่า Sidebar ย่อ-ขยายได้อย่างราบรื่น
3. กดปุ่ม Toggle Sidebar ในส่วนหัว ตรวจสอบว่าแผงไลบรารียุบและขยายได้อย่างนุ่มนวล
4. ตรวจสอบว่าเพลย์ลิสต์เมื่อเปลี่ยนมุมมองการ์ด จะเรียงตัวเป็นตารางรูปหน้าปก (Thumbnail Grid) คล้ายคลึงกับรูปภาพต้นแบบสากล
5. เปิดแท็บเพลงในเพลย์ลิสต์หรือแท็บค้นหาเพลง จากนั้นกดสลับปุ่ม List/Grid ตรวจสอบว่าแทร็กเพลงไหลออกมาเป็นรูปหน้าปกจัตุรัสย่อ และยืดสุดความสูงของหน้าจอโดยสมบูรณ์
6. ตรวจสอบรูปภาพรอบระบบว่าโค้งมนในระดับ `rounded` ทั่วไป
